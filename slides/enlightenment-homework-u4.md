4) 编写考鼎码算法，将给定的分数（依次输入分子和分母）化为最简分数形式，分别输出其整数部分以及最简分数的分子及分母。

```mathematica
# 假定 a, b 均为正整数
算始 化简分数 (a, b)
    n := 0          # 整数部分
    若始 (a > b)
        n := a // b
        a := a % b
    若终

    # 求最大公约数
    x : = a; y := b
    当始 (y != 0)
        tmp := x
        x := y
        y := tmp % y
    当终

    a := a // x
    b := b // x

    输出 n
    输出 a
    输出 b
算终
```
