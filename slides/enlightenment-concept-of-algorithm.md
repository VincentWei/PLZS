## 进制转换算法

- 初始算法描述：

```console
算法开始 转换进制 (x, b)
    循环开始 (当 x ≠ 0 时)
        令 r = x % b
        令 x = x // b
        输出 r
    循环结束
算法结束
```

	
- 分析这个算法的问题

```
算法开始 转换进制 (x, b)
    令 s 为一个空的序列
    如果 (x = 0)
        将 0 放入 s
        返回 s
    如果结束

    循环开始 (当 x ≠ 0 时)
        r = x % b
        将 r 插到 s 的最前面
        令 x = x // b
    循环结束

    返回 s
算法结束
```
