## 背景知识

- 常见桌面操作系统
   - Windows
   - macOS
   - Linux 桌面系统
- 流行的 Linux 发行版
   - Debian（开源社区维护）
   - RedHat（红帽公司的商业版本）
   - Ubuntu（基于 Debian 的商业版本）
- Ubuntu Linux 发行版的版本号
   - 版本号由首次发布年份及月份组成，如 22.04, 23.10 等。
   - 每年出两个版本，如 22.04、22.10、23.04、23.10 等。
   - 每两年出一个长期支持（LTS）版本，如 20.04、22.04、24.04 等。
   - 除了 LTS 之外的版本是实验性的版本，适用于喜欢尝鲜的用户。
